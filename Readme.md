This is a Git practice repository
Branching
Create a new folder `git-practice` and initialize a new Git repository.
Create a new file `README.md` and add some text to it
Add the file to the staging area.
Make a commit with the message "Add README.md"
Create a new branch `feature-1`.
Switch to the new branch.
Modify the file and make a commit with the message "Update README.md in feature-1".
Switch back to the `main` branch.
Merge the `feature-1` branch into the `main` branch.
